Политика конфиденциальности

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок сбора, использования, хранения и защиты персональных данных пользователей Telegram-бота (далее — «Бот»), предоставляемого MMC.co. Используя Бот, пользователь выражает своё согласие с условиями данной Политики.

2. Состав и категории обрабатываемых данных

Бот может получать и обрабатывать следующие данные:

  • Уникальный идентификатор пользователя Telegram (user_id)
  • Имя пользователя (username), если указано
  • Текстовые сообщения, отправленные Боту
  • Файлы (документы, изображения, видео), отправленные Боту
  • Технические данные (дата и время обращений, служебные метки)

При использовании функций пересылки сообщений через WhatsApp также обрабатываются:

  • Временные данные WhatsApp-сессии, необходимые для установления и поддержания сессии
  • Передаваемые файлы и текст, направленные пользователем через Бот

3. Цели обработки данных

Собранные данные используются исключительно для:

  • Обеспечения работоспособности Бота и предоставления его сервисов
  • Пересылки сообщений и файлов через WhatsApp
  • Обеспечения безопасности и предотвращения злоупотреблений
  • Обратной связи с пользователем
  • Проведения статистических и аналитических исследований в обезличенной форме

4. Передача данных третьим лицам

Данные передаются третьим лицам только в следующих случаях:

  • Техническая передача данных и файлов для доставки сообщений через WhatsApp
  • Доступ к данным WhatsApp-сессии имеют только уполномоченные администраторы MMC.co и исключительно в случаях, предусмотренных законом

5. Хранение и защита данных

Данные хранятся в объёме и сроках, необходимых для работы Бота. Меры защиты:

  • Шифрование хранения для чувствительных данных
  • Защищённые каналы связи (TLS)
  • Ограничение доступа по ролям
  • Логирование действий администраторов и аудит
  • Регулярное обновление серверов

6. WhatsApp-сессии

Бот использует временные WhatsApp-сессии для пересылки сообщений. Сессии используются только для доставки сообщений.

7. Права пользователя

Пользователь имеет право:

  • Получать информацию о своих данных
  • Удалять данные
  • Отозвать согласие на обработку данных

8. Изменение политики

Политика может быть обновлена. Актуальная версия доступна в описании бота в Telegram.
